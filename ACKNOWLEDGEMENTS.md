# Acknowledgements & Credits

Vantage is an **orchestrator**. It does not contain, bundle, redistribute, or
modify any of the tools below — it simply **invokes them as separate programs**
that you install on your own machine, and formats their output into one report.
All copyrights and trademarks belong to their respective owners, and each tool
is used under its own license. Vantage would be nothing without this incredible
open-source ecosystem — **thank you** to every author and contributor. 🙏

If you use Vantage, please support and star the upstream projects.

> Note on licensing: because Vantage merely calls these programs over their
> command-line interfaces (the normal way any user would run them), it does not
> create a derivative work of them and does not relicense them. Install each from
> its official source; its own license and terms apply to your use of it.
